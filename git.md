# Useful git commands

### Update git
PS>git update-git-for-windows

### View git configuration
PS>git config --list

### Configure name and email address for commits
PS>git config [--global] user.name "Full Name"
PS>git config [--global] user.email "email@address.com"


### View remote origin
PS>git remote -v


PS>git branch
